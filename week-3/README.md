# Week 3: Process Domain Part 1 — Integration, Scope, Schedule, Cost/EVM

## Why This Week Matters

This is the **heaviest technical content** in your PMP study plan. The Process domain accounts for **50% of the exam** (approximately 90 out of 180 questions), and this week covers the four knowledge areas most likely to produce calculation questions and complex scenarios. Integration Management ties everything together, Scope defines what you're building, Schedule determines when, and Cost/EVM measures performance. Master these and you've conquered the hardest quantitative content on the exam.

## Learning Objectives

By the end of this week, you should be able to:

1. Explain all 7 Integration Management processes and when each is performed
2. Walk through the Integrated Change Control process step-by-step (CCB workflow)
3. Create a WBS using decomposition, apply the 100% rule and 8/80 rule
4. Distinguish between Validate Scope and Control Quality
5. Calculate the critical path, total float, and free float on a network diagram
6. Apply schedule compression techniques (crashing vs. fast-tracking) with trade-off analysis
7. Perform all EVM calculations (SV, CV, SPI, CPI, EAC, ETC, VAC, TCPI)
8. Select the correct EAC formula based on project conditions
9. Interpret EVM results and make project decisions based on variances and indices

## Key Themes for the Process Domain (Part 1)

- **Integration is the PM's unique responsibility** — no one else does this work
- **Change requests go through ICC** — never make changes without formal approval
- **The WBS is the foundation** — if it's not in the WBS, it's not in the project
- **Critical path = longest path** — it determines minimum project duration
- **EVM is objective** — it removes opinion from project status reporting
- **CPI rarely improves** — once cost performance degrades, it typically stays degraded

## Calculation Warning

This week contains the most math-heavy content in the entire study plan. You MUST be able to:

- Calculate forward/backward pass on network diagrams
- Compute all EVM metrics from raw PV, EV, AC data
- Select the right EAC formula and justify your choice
- Interpret negative/positive variances and indices above/below 1.0

Practice these calculations until they're automatic. On exam day, you won't have time to derive formulas — they need to be memorized.
