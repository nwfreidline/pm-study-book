## What Is This?

A self-contained HTML study book, quiz app, and visual cheat sheet collection covering everything you need to pass the PMP exam. No account required, no paywall, no ads. Just open it in a browser and start reading.

Built from the PMBOK Guide 7th Edition, the PMI Exam Content Outline, and the most recommended community study resources.

**Design:** PMI-inspired navy/gold color theme with dark mode toggle on every page.

## 🚀 Crash Course — Final Review

29 pages covering the entire exam in condensed format: all domains, all formulas, all frameworks, visual diagrams, and exam tips. Designed for the final 2-3 days before the exam.

## 🎯 Practice Exam Simulator

Full-length 180-question simulation mirroring the real exam: 230-minute timer, breaks at Q60/Q120, question flagging, pass/fail scoring with domain proficiency levels, and complete answer review.

## How to Use

### Online (Recommended)
Visit **[nwfreidline.github.io/pm-study-book](https://nwfreidline.github.io/pm-study-book/)** — works on any device.

### Offline
Download or clone this repo and open `index.html` in any browser.

```
index.html              ← Study book (39 pages)
practice-exam.html      ← Exam simulator (180Q, timed)
crash-course/           ← Crash course (29 pages)
quiz.html               ← Practice quiz (405Q, filtered)
quiz-bank.js            ← Question bank (255 base)
quiz-bank-expansion.js  ← Question bank expansion (+150)
cheat-sheets/           ← 12 cheat sheets + print version
week-1/                 ← Week 1 materials
week-2/                 ← Week 2 materials
docs/                   ← Reference materials
```
